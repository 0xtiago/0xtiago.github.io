Nesta vamos falar sobre concatenar strings (juntar strings), interpolação de strings (colocar valores dentro de strings) e processar a entrada do usuário. Também vamos falar sobre como podemos documentar melhor nosso código usando comentários.

Concatenação de string

Vamos começar com o reasultado do exercício da aula anterior:

valor_hora = input("Digite o valor da sua hora de trabalho: ")
horas_trabalhadas = input("Quantas horas você trabalhou esta semana? ")

print("Valor hora: ")
print(valor_hora)

print("Horas trabalhadas: ")
print(horas_trabalhadas)

Aqui, pedimos ao usuário duas informações e, em seguida, imprimimos essas informações de volta no console junto com alguns cabeçalhos.

Isso é muito bom, mas não seria bom se pudéssemos ter o valor valor_hora na mesma linha que o rótulo "Valor hora: "? É aqui que entra a concatenação de strings (juntando duas ou mais strings).

Concatenar coisas significa juntar ou ligar coisas, e é isso que queremos fazer com nossas strings. Queremos pegar a string, "Valor hora: ", e queremos juntar a ela a string que atribuímos à variável,valor_hora.

Em Python, podemos apenas usar o operador + para isso, assim:

valor_hora = input("Digite o valor da sua hora de trabalho: ")
horas_trabalhadas = input("Quantas horas você trabalhou esta semana? ")

print("Valor hora: " + valor_hora)
print("Horas trabalhadas: " + horas_trabalhadas)

Muito parecido com quando usamos + para somar números, podemos encadear quantas strings quisermos usando +. No entanto, devemos ter muito cuidado para não combinar os tipos.

Por exemplo, não podemos fazer algo assim, em que tentamos juntar um inteiro em uma string:

print("Valor hora: " + valor_hora)

Se executarmos este programa, o Python retornará o seguinte erro:

Desta vez, obtemos um TypeError, que nos diz que tentamos concatenar um int (inteiro) em uma str (string), mas esta operação só é suportada entre strings.

Portanto, a questão agora é: o que faremos se precisarmos realizar algumas operações aritméticas e, em seguida, colocar o resultado em uma string? Felizmente, o Python nos fornece algumas ferramentas para converter valores de um tipo em outro.

Convertendo strings, inteiros e floats

Primeiro, vamos falar sobre a conversão de inteiros e flutuantes em strings. A ferramenta que usaremos é a chamada str, e podemos pensar nela como sendo muito parecida com uma função.

Quando chamamos str, passando um inteiro ou um float, vamos obter uma nova string contendo os numerais que compõem o número, junto com as casas decimais no caso de floats.

Por exemplo, podemos escrever algo assim:

idade = str(12)

O que vai atribuir à idade a mesma coisa que se escrevêssemos:

idade = "12"

Converter de uma string em um inteiro ou float é um pouco mais complicado, porque nem toda string é um número válido. Além disso, mesmo se uma string for um número válido, pode não ser um número válido do tipo para o qual estamos tentando converter.

Se quisermos converter uma string em um inteiro, precisamos chamar int, passando a string. Os caracteres da string devem representar um número inteiro.

idade = int("12") ⚠️

Se tentarmos passar uma representação de string de um número com casas decimais, vamos obter um TypeError. Por exemplo, algo assim:

valor_hora = int("20.53") ⚠️

Ao executar retornará a seguinte mensagem de erro:

Traceback (most recent call last):
    File "main.py", line 1, in <module>
        int("20.53")
ValueError: invalid literal for int() with base 10: '20.53'

Ao invés de int, necessitamos chamar float:

valor_hora = float("20.53") ✔️

float também pode manipular a entrada sem um componente decimal. Por exemplo, se passarmos "20", vamos obtê-lo com casas decimais 20.0 de volta.

Além de converter de strings em inteiros e flutuantes, e vice-versa, também podemos converter entre inteiros e flutuantes usando o mesmo int e float. Neste caso, passar um float para int vai funcionar, mas vai “truncar” o float, essencialmente jogando fora tudo após a vírgula decimal.

Truncamento é o ato de encurtar algo removendo alguma parte dele. Por exemplo, podemos nos referir a truncar um float, o que significa que estamos removendo tudo após o ponto decimal.

Interpolação de strings com o método format

A concatenação de strings é uma coisa útil de se saber, mas é um pouco desajeitada e está sujeita a erros de espaçamento, como vimos. Uma abordagem melhor costuma ser usar a interpolação de strings, especialmente ao trabalhar com conteúdo que tem uma estrutura semelhante a uma frase.

A interpolação de string é o ato de inserir algum conteúdo novo em uma string existente.

Ao contrário da concatenação, a interpolação depende de colocarmos marcadores em uma string que podemos preencher com valores. Isso torna muito mais difícil cair em problemas de formatação, porque os marcadores de posição estão na posição exata que os valores irão ocupar. Outro grande benefício da interpolação: não precisamos nos preocupar com os tipos! Python irá garantir que obteremos uma representação de string de todos os nossos valores.

Existem várias maneiras de realizar a interpolação de string em Python, mas primeiro veremos como usar o método de formatação format.

O método format possuí diversas formas de ser utilizado, utilizaremos a forma simplificada dele. 😉

Ao usar o método format, os marcadores de posição para nossos valores são chaves, que se parecem com isto: {}. Por exemplo, digamos que queremos escrever uma string como:

"Tiago tem 35 anos!"

Mas em vez de apenas "Tiago" e "35", queremos ser capazes de escrever qualquer nome e qualquer idade aqui. Tudo o que temos que fazer é substituir "Tiago" e "35" por chaves, assim:

f"{} tem {} anos!"

Usando este exemplo, nosso programa ficaria assim:

nome = "Tiago"
idade = 35

print(f"{nome} tem {idade} anos!")

Podemos também utilizar o método para escrever expressões dentro dentro dele:

nome = "Tiago"
idade = 35

print(f"{nome} tem {idade*12} meses de idade!")

Comentários

Usando nomes bons e descritivos, podemos fazer muito para tornar nosso código autodocumentado; no entanto, às vezes, mesmo um bom código pode ser difícil de entender. Afinal, algumas coisas são complicadas.

Em casos como este, pode ser útil usar comentários para explicar aos leitores de nosso código (o que nos inclui) o que o código está fazendo.

Para escrever um comentário, só precisamos colocar um # diretamente antes da mensagem que queremos escrever. Isso dirá ao Python para não tratar o que se segue como código.

nome = "Tiago"
idade = 35

#Imprime a idade em meses
print(f"{nome} tem {idade*12} meses de idade!")

Tente não exagerar nos comentários, pois comentários em todas as linhas podem causar mais danos do que benefícios no que diz respeito à legibilidade.

Processamento básico de strings

A última coisa que é bom mencionar nesta aula são algumas operações comuns de processamento de strings, como alterar a caixa das letras ou remover espaços em branco (caracteres de espaço, tabulações, novas linhas, etc.) das extremidades das strings. Esse tipo de coisa acontece o tempo todo quando lidamos com a entrada do usuário.

Temos várias opções diferentes para alterar a caixa das letras em uma string. Aqui, vamos nos concentrar em quatro opções importantes: lower, upper, capitalize e title.

lower e upper transformam a string inteira em minúsculas e maiúsculas, respectivamente. Os caracteres que não possuem maiúsculas e minúsculas, como caracteres de pontuação, são ignorados.

capitalize vai transformar o primeiro caractere em maiúsculas, com o resto em minúsculas. title vai transformar a string em maiúsculas, o que significa que cada palavra começa com uma letra maiúscula e todas as outras letras são transformadas em minúsculas.

Para utilzar estes métodos, escrevemos da seguinte forma:

"Olá, Mundo!".lower()       # "ola, mundo!"
"Olá, Mundo!".upper()       # "OLÁ, MUNDO!"
"Olá, Mundo!".capitalize()  # "Olá, mundo!"
"Olá, Mundo!".title()       # "Olá, Mundo!"

Em cada caso, estamos escrevendo uma expressão, e o valor dessa expressão é uma nova string no caso solicitado.

Não há muito mais a fazer quando se trata de mudar de caixa, mas que tal remover o espaço em branco no fim das strings? Para isso, podemos usar o método strip.

strip é na verdade muito mais versátil do que isso, e podemos fazer com que ele remova qualquer coisa que quisermos, mas por padrão ele nos dará uma string nova com qualquer espaço em branco estranho removido de ambas as extremidades.

Nós o usamos exatamente da mesma maneira que lower, upper, etc.

"    Olá, Mundo!    ".strip() # "Olá, Mundo!"

Perceba no exemplo acima como os espaços das extremidade são removidas.

Processando múltiplas linhas

Podemos aplicar vários métodos de processamento a uma string. Deixe-me mostrar duas maneiras de fazer isso.

A primeira forma envolve a criação de uma variável e a reatribuição a ela para cada chamada de método, assim:

user_name = " ANA OLIVEIRA  "
user_name = user_name.strip()  # "ANA OLIVEIRA"
user_name = user_name.title()  # "Ana Oliveira"

Embora isso seja muito repetitivo, e reatribuir a uma variável várias vezes geralmente pareça ineficiente (embora não seja lento, Python pode reatribuir a variáveis muito rapidamente).

Portanto, a segunda maneira de fazer isso envolve usar o resultado de cada etapa em linha, assim:

user_name = " ANA OLIVEIRA  ".strip().title()

Aprenderemos mais sobre como isso funciona exatamente quando examinarmos as funções e os valores de retorno, mas por enquanto tratamos cada chamada de método, como "ANA OLIVEIRA".strip(), como a string que ela devolveria a você.

Exercícios

  1. Utilizando a variável abaixo, imprima "Olá , mundo!":
 saudacao = "Olá, mundo"

Você pode adicionar o ponto de exclamação ausente usando concatenação de string ou format. A escolha é sua.

  1. Peça o nome do usuário e, em seguida, cumprimente-o, usando o nome dele como parte da saudação. O nome deve estar em caixa do título (title) e não deve ser circundado por nenhum espaço em branco.

    Por exemplo, se o usuário digitar “eduardo”, sua saída deve ser algo assim:

    Olá, Eduardo!

  2. Concatene a string "Eu tenho" , inteiro 48, e novamente outra string "anos." para pruduzir uma string "Eu tenho 48 anos." Lembre-se de que só podemos concatenar strings para outras strings, portanto, você terá que converter o inteiro em uma string antes de realizar a concatenação.

  3. Formate e imprima as informações abaixo utilizando interpolação de strings:]

    titulo = "O Coringa"
diretor = "Todd Phillips"
ano = 2019

    O resultado deve parecer como o abaixo:

    O Coringa (2019), dirigido por Todd Phillips.
