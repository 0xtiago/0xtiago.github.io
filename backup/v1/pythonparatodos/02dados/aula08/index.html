Agora vamos voltar às coleções e vamos falar principalmente sobre a conversão entre strings e outros tipos de coleção, como listas e tuplas. Vamos aprender como pegar um subconjunto de uma coleção usando o fatiamento.

Convertendo tuplas e listas para strings

Tuplas e listas têm uma representação de string, mas muitas vezes não é o que queremos. Se tentarmos passar uma lista para str para criar uma nova string, assim:

numeros = [1, 2, 3, 4, 5]
numeros = str(numeros)

Veremos algo assim na saída:

[1, 2, 3, 4, 5]

Você pode ter notado que isso é exatamente o que obteremos se tentarmos imprimir uma lista de qualquer maneira. Isso ocorre porque quando passamos uma lista paraprint, print converte a lista em uma string para nós.

Em termos de código, é isso que você possui, uma string contendo colchetes, números e vírgulas:

numeros = "[1, 2, 3, 4, 5]"

Neste caso, o que queremos é isso:

1,2,3,4,5

Por exemplo, talvez eu tenha uma lista de nomes que descreve quem trabalhou em algum projeto e quero imprimir esses nomes como parte de uma frase:

As pessoas que trabalharam neste projeto são: Juca, Maria, Alberto.

O que eu não quero quer aconteça é o seguinte:

As pessoas que trabalharam neste projeto são: [Mike, Sofia, Helen].

A solução para este problema é utilizar o método join.

Juntando itens de uma coleção com o método join

Chamamos join em uma string, e essa string é o que queremos colocar entre os itens da coleção que queremos colar. No exemplo acima, queremos apenas juntar os valores usando uma vírgula e um espaço, para que nosso código fique assim:

colaboradores = ['Mike', 'Sofia', 'Helen']
colaboradores = ", ".join(colaboradores)

print(f"As pessoas que trabalharam neste projeto são: {colaboradores}.")

Poderíamos conseguir o mesmo colocando a chamada join dentro das chaves da f-string:

colaboradores = ['Mike', 'Sofia', 'Helen']

print(f"As pessoas que trabalharam neste projeto são: {', '.join(colaboradores)}.")

Apenas tome cuidado com suas frases! Se você usar as mesmas aspas para definir a string do join, vai acabar cortando a string antes do desejado.

Uma coisa que você deve estar ciente ao usar join é que só podemos unir coleções de strings. Se temos uma lista de números, primeiro temos que converter cada número em uma string.

  • Começamos com uma lista de números, chamados de números.
  • Em seguida, criamos uma lista vazia, numeros_em_string. Isso manterá nossos números convertidos em strings.
  • Usando um loop for, iteramos sobre a lista de números. Convertemos cada número em uma string e o acrescentamos a numeros_em_string.
  • Finalmente, usamos join com numeros_em_string.
numeros = [1, 2, 3, 4, 5]
numeros_em_string = []

for numero in numeros:
	numeros_em_string.append(str(numero))

print(', '.join(numeros_em_string)) # 1, 2, 3, 4, 5

Parece realmente complicado, mas existem maneiras muito mais eficientes e sucintas de fazer algo como isso que exploraremos mais tarde.

Dividindo uma strings ou textos com split

Muitas vezes, em vez de combinar uma coleção em uma única string, queremos converter uma string em outra coleção de vários itens. Por exemplo, podemos desejar fazer isso ao processar a entrada do usuário se o usuário fornecer vários valores em resposta a um único prompt.

Uma maneira comum de fazer isso é usar o método split. Ao chamar split, geralmente precisamos passar uma segunda string que usaremos como sequência separadora. Sempre que essa sequência de caracteres é encontrada, isso marca o fim de um valor e o início do próximo.

Por exemplo, digamos que queremos cinco números do usuário e solicitamos que esses números sejam separados por vírgulas. Podemos escrever algo assim:

numeros = input("Por favor, digite 5 números separados por virgula: ") # 1,2,3,4,5
numeros_lista = numeros.split(",")

print(numeros_lista) # ['1', '2', '3', '4', '5']

Aqui, especificamos uma vírgula como a sequência separadora, de modo que cada vírgula marca o final de um valor que queremos adicionar à coleção.

Em vez disso, poderíamos pedir ao usuário para inserir números separados por traços e, em seguida, teríamos que passar o traço para o split como separador:

numeros = input("Por favor, digite 5 números separados por virgula: ") # 1-2-3-4-5
numeros_lista = numeros.split("-")

print(numeros_lista) # ['1', '2', '3', '4', '5']

O que recebemos da divisão é uma lista, mas sempre podemos criar outro tipo de coleção, passando o resultado para tupla, por exemplo, se é isso que queremos:

numeros = input("Por favor, digite 5 números separados por virgula: ") # 1,2,3,4,5
numeros_tupla = tuple(numeros.split(","))

print(numeros_tupla) # ('1', '2', '3', '4', '5')

Uma coisa a ter em conta é que a divisão não vai tirar nenhum espaço em branco para nós. Se o usuário escrever os números com vírgulas seguidas de espaços, esses espaços acabarão nas cadeias de caracteres que colocamos na nova coleção:

numeros = input("Por favor, digite 5 números separados por virgula: ") # 1, 2, 3, 4, 5
numeros_lista = numeros.split(",")

print(numeros_lista) # ['1', ' 2', ' 3', ' 4',  ' 5']

Removendo espaços em brancos com strip

Às vezes, isso não é realmente um problema, mas às vezes você pode precisar percorrer a coleção com um loop for e limpar as coisas com o strip.

numeros = input("Por favor, digite 5 números separados por virgula: ") # 1, 2, 3, 4, 5
numeros_lista = numeros.split(",")
numeros_lista_limpa = []

for numero in numeros_lista:
    numeros_lista_limpa.append(numero.strip())

print(numeros_lista_limpa) # ['1', '2', '3', '4', '5']

Você pode usar o mesmo tipo de abordagem para processar as strings individuais de outras maneiras também.

Se não especificarmos uma sequência para o split, ela será dividida com base em qualquer comprimento de espaço em branco. Isso é muito útil, porque sabemos que, independentemente do que recebermos, não teremos que nos preocupar em removê-lo. O usuário pode inserir algo assim:

1 2 3 4 5

E ainda teríamos uma lista como esta:

['1', '2', '3', '4', '5']

Isso também funciona se estivermos trabalhando com strings divididas em linhas diferentes.

Nem sempre precisamos chamar o split. Se quisermos apenas colocar cada caractere como um item diferente em uma lista ou tupla, podemos apenas passar a string para a função de lista (list) ou tupla(tuple):

string_simples = "Python"

print(list(string_simples)) # ['P', 'y', 't', 'h', 'o', 'n']
print(tuple(string_simples)) # ('P', 'y', 't', 'h', 'o', 'n')

Quabrando linhas

Uma coisa que você pode estar se perguntando é: como podemos dizer ao computador que devemos ter uma quebra de linha?

Na verdade, há um caractere invisível que é adicionado quando marcamos o final de uma linha pressionando “Enter”. Se você entrar em um processador de texto e pressionar “Enter” para ir para a próxima linha, o que acontece quando você pressionar “Backspace”? Excluímos algo e isso nos leva de volta à linha anterior. O que você excluiu é o caractere de nova linha que marca o fim da linha.

No Python, representamos esse caractere como \n e podemos usá-lo em strings como qualquer outro caractere. Por exemplo, posso escrever uma string como esta:

print("Programa super especial\n\nPor João Roberto")

Teríamos uma saída como essa:

Programa super especial

Por João Roberto

Se quisermos, podemos juntar strings usando este caractere, e é possível dividir strings usando \n também, mas isso não é realmente útil para entrada do usuário. No entanto, pode ser muito útil ao trabalhar com arquivos, que é algo que abordaremos mais a frente.

Removendo espaços em branco com replace

Caso queremos tratar letra por letra em uma string, pode utilizar o método replace para substituir espaços em branco por nada, ou por qualquer outro caractere.

texto = input("Digite um texto: ")
tamanho_do_texto = len(list(texto.replace(" ",""))) 

print(f"O texto '{texto}' possui {tamanho_do_texto} letras.")

No exemplo acima, obtemos um texto digitado pelo usuário, e como método replace, removemos os espaços em branco (" ") por vazio ("").

Fatiando textos com slice

Até agora, vimos como separar strings e juntar os valores de outras coleções, mas e se quisermos apenas pegar alguma parte de uma coleção? Para isso, podemos usar o fatiamento.

Vejamos um exemplo com uma string. Digamos que temos a string Python, e eu quero os primeiros 3 caracteres dessa string por algum motivo. Usando uma fatia, escreveríamos isto:

string_original = "Python"
string_fatidada = string_original[0:3]

print(string_fatiada)  # Pyt

Aqui nossa fatia foi escrita como [0:3], o que significa que queremos começar no índice 0 (Primeira letra), e queremos pegar todos os itens, mas não incluindo o índice 3(letra ‘h’). O que recebemos de volta é uma nova string contendo apenas três caracteres : "Pyt".

Há outra maneira pela qual poderíamos ter escrito esta fatia, que é assim: [:3]. Se não especificarmos um valor para o índice inicial, a fatia começa no início da sequência. Se não especificar um índice de parada, a fatia para no final da sequência.

Então, se escrevemos isso [3:], isso significa me dê tudo do índice 3 em diante. Podemos ver um exemplo aqui:

string_original = "Python"
string_fatidada = string_original[3:]

print(string_fatiada)  # hon

Lá na aula 4, vimos que podemos usar índices negativos para acessar elementos em uma sequência e também podemos usar índices negativos com fatias. No entanto, é muito importante que você entenda que [3:-1] não é a mesma coisa que [3:], porque com fatias, o índice de parada não é inclusivo.

Quando escrevemos [3: -1], estamos dizendo que queremos tudo, desde o índice 3 até, mas não incluindo, o elemento final. Quando escrevemos [3:], estamos dizendo que queremos tudo do índice 3 em diante. Podemos ver a diferença aqui:

string_original = "Python"

print(string_original[3:])  # hon
print(string_original[3:-1])  # ho

Se você quiser copiar uma coleção inteira por algum motivo, podemos realmente pegar uma fatia contendo toda a sequência usando [:].

Fatias são uma ferramenta incrivelmente versátil e há muito mais para aprender.

A função len

Se quisermos descobrir quantos itens há em uma determinada coleção, podemos usar a função len, abreviada para “lenght”, comprimento em português.

Isso funcionará para strings, tuplas, listas e qualquer um dos outros tipos que examinaremos posteriormente neste curso. Se você tentar chamar len em um tipo que não suporta esta operação, você receberá uma exceção como esta:

Traceback (most recent call last):
  File "main.py", line 1, in <module>
    len(12)
TypeError: object of type 'int' has no len()

Supondo que passe um tipo apropriado para len, ele retornará um inteiro que representa o número de itens na coleção:

numeros = [1, 2, 3, 4, 5]
len(numeros) # 5

Exercícios

  1. Peça ao usuário para inserir seu nome e sobrenome em resposta a uma única solicitação. Use split para extrair os nomes e, em seguida, atribua cada nome a uma variável diferente. Para este exercício, você pode supor que o usuário tenha um único nome e um único sobrenome.

  2. Imprima a lista, [1, 2, 3, 4, 5], no formato 1 | 2 | 3 | 4 | 5 usando o método join. Lembre-se de que você só pode juntar coleções de strings, então você vai precisar fazer algum processamento inicial da lista de números.

  3. Abaixo você encontrará uma pequena lista de citações:

    frases = [
    "'Sábio é aquele que conhece os limites da própria ignorância..'",

    "'Às vezes ouço passar o vento; e só de ouvir o vento passar, vale a pena ter nascido.'",

    "'Aceita o conselho dos outros, mas nunca desistas da tua própria opinião.'",

    "'A mentira nunca vive o suficiente para envelhecer.'"
]

    Cada citação é uma string, mas na verdade cada string contém caracteres de aspas no início e no final. Usando o slice, extraia o texto de cada string, sem essas aspas extras, e imprima cada citação.

    Você também pode tentar uma solução usando strip.

  4. Parte 1: Peça ao usuário para inserir uma palavra e, em seguida, imprima o comprimento da palavra. Você deve levar em conta qualquer excesso de espaço em branco na entrada do usuário, então você terá que processar a string antes de encontrar seu comprimento.

    Parte 2: Se você quiser ir um pouco mais longe, peça ao usuário um longo pedaço de texto. Você pode então dizer a eles quantos caracteres existem no texto geral, e você também pode fornecer a eles uma contagem de palavras.
